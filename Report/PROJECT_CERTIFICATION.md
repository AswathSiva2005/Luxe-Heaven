## PROJECT ABSTRACT

**LUXE HAVEN** is a comprehensive full-stack e-commerce and product management platform developed to streamline online retail operations. The system encompasses features including:

- **Product Management**: Comprehensive product catalog with detailed information, pricing, and inventory management
- **Shopping Cart & Checkout**: Complete user shopping experience with cart management and secure checkout
- **Order Management**: Order processing, tracking, and history for both buyers and sellers
- **User Authentication**: Secure user registration, login, and profile management
- **Seller Dashboard**: Tools for sellers to manage their products and orders
- **Wishlist Management**: Personalized wishlist functionality for users
- **Stock History**: Detailed tracking of inventory changes and stock levels
- **Discount Configuration**: Dynamic discount and promotional pricing system
- **Support System**: Customer support ticket system for user assistance

### Technology Stack

**Frontend:**
- React with Vite
- Tailwind CSS for responsive design
- Modern JavaScript (ES6+)

**Backend:**
- Node.js with Express
- MongoDB database
- JWT-based authentication
- RESTful API architecture
